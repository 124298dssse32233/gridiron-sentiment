TYPES & INTERFACES
================================================================================

All OG functions use strongly typed data:

TeamOGData
├─ teamName: string
├─ mascot?: string
├─ rating: number
├─ ratingDeviation?: number
├─ rank: number
├─ level: 'FBS' | 'FCS' | 'D2' | 'D3' | 'NAIA'
├─ conference?: string
├─ primaryColor?: string
├─ secondaryColor?: string
└─ record?: string

RankingsOGData
├─ week?: number
├─ season: number
└─ topTeams: Array<{
   ├─ rank: number
   ├─ name: string
   ├─ rating: number
   └─ level: string
}

MatchupOGData
├─ teamA: { name, rating, primaryColor? }
├─ teamB: { name, rating, primaryColor? }
├─ winProbA: number
├─ spread?: number
└─ date?: string

ChaosOGData
├─ gameTitle: string
├─ chaosScore: number
├─ chaosTier: 'LEGENDARY' | 'EPIC' | 'CHAOS' | 'WEIRD' | 'NORMAL'
├─ score: string
└─ date?: string

ArticleOGData
├─ title: string
├─ subtitle?: string
├─ date: string
└─ category?: string

================================================================================
QUICK START
================================================================================

1. Add to layout.tsx:
   import { Metadata } from 'next';
   
   export const metadata: Metadata = {
     icons: {
       icon: '/favicon.svg',
       apple: '/icon.svg',
     },
   };

2. Generate OG image:
   import { generateTeamOG } from '@/lib/utils/og-image';
   
   const svg = generateTeamOG({
     teamName: 'Alabama',
     rating: 1547,
     rank: 1,
     level: 'FBS',
   });

3. For dynamic metadata:
   export async function generateMetadata() {
     const svg = generateTeamOG(data);
     return {
       openGraph: {
         images: [{
           url: `data:image/svg+xml;base64,${btoa(svg)}`,
           width: 1200,
           height: 630,
         }],
       },
     };
   }
